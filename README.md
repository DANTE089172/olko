# OLKO

Strategic Sourcing. Intelligent Supply. Real Results.

Marketing site for OLKO — a polymer procurement and supply intelligence company.

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on Cloudflare Pages

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy to Cloudflare Pages

```bash
npm run deploy
```

Or connect the repo to Cloudflare Pages with build command `npx @cloudflare/next-on-pages` and output directory `.vercel/output/static`.
